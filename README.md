# Dotfiles

to install these dotfiles, you use gnu stow. These dotfiles aim for a keyboard-centric minimalist macos.

## install gnu stow

```bash
brew install stow
```

## clone this repo

```bash
cd ~
git clone https://github.com/igor-mauricio/.dotfiles.git
```
## stow the dotfiles 

```bash
cd .dotfiles
stow */
```
Enjoy!
